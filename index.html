"Anhel" es una ambiciosa aplicación web que nace con la misión de mejorar la salud mental al proporcionar servicios psicológicos online de manera accesible y anónima, si así lo prefiere el usuario.
Anhel permite a los usuarios participar en chats privados con profesionales. La plataforma incluye un área dedicada a la lectura de blogs y publicaciones de psicólogos.
Anhel cuenta con un sistema de registro y autenticación.
Los usuarios pueden llevar un control detallado de su progreso y pensamientos a través de un sistema de notas personalizado.
Si el usuario decide dar un paso más allá del anonimato, Anhel ofrece la posibilidad de revelar información al psicólogo

Mi Funcion:

Mi contribución abarcó tanto la maquetación y el diseño de las interfaces como la implementación de funcionalidades utilizando el framework Angular.
Desarrollé los servicios en Angular necesarios para facilitar la comunicación efectiva entre el frontend y el backend. Esto incluyó la implementación de lógica para la gestión de sesiones y el manejo de datos

Descripción:

"Creditcar" es una plataforma web diseñada como un intermediario efectivo para conectar a clientes en busca de vehículos con concesionarios que ofrecen financiamiento "in house". El objetivo principal es simplificar la búsqueda de vehículos para los clientes, al tiempo que brinda a los concesionarios una oportunidad de llegar a nuevos compradores y aumentar sus ventas.
La plataforma permite a los clientes seleccionar vehículos de interés y contactar directamente a los concesionarios para obtener más detalles o concertar visitas para ver y probar los vehículos. "Creditcar" actúa como un facilitador de comunicación, dejando que clientes y concesionarios negocien los términos de venta sin gestionar transacciones financieras directas.

Mi Funcion:

Mi contribución al proyecto "Creditcar" se centró en el desarrollo del backend, utilizando Node.js y Express.js para crear una API robusta que impulsara la funcionalidad integral de la plataforma. Los endpoints proporcionaron la funcionalidad necesaria para la gestión de vehículos, ofertas de concesionarios y la interacción entre clientes y concesionarios.

Desarrollé un algoritmo personalizado que evaluaba la probabilidad de que un concesionario otorgara crédito a un usuario. Este algoritmo, basado en datos del usuario y requisitos del concesionario, permitía ofrecer recomendaciones altamente personalizadas sobre vehículos, ahorrando tiempo a los usuarios al evitar solicitudes de crédito improbables. Y Establecí la conexio con la base de datos MongoDB y cree sus respectivas schemas.

Descripción:

"Worrdlo" surge como respuesta a la necesidad de adaptarse a los desafíos educativos que surgieron durante la pandemia y el consecuente confinamiento de los estudiantes. La idea central detrás de este proyecto fue crear una plataforma integral que digitalizara herramientas esenciales, facilitando así el aprendizaje virtual en nuestra institución educativa.
"Worrdlo" no solo buscaba digitalizar herramientas educativas, sino crear un entorno virtual que promoviera la colaboración, la comunicación efectiva y la organización en el ámbito académico.

Mi Funcion:

En el proyecto "Worrdlo", asumí el rol de desarrollador full stack, implementando dos componentes fundamentales: el sistema de notas (apuntes) y el sistema de calendario (agendamiento de actividades). Mi enfoque abarcó tanto el frontend, donde utilicé Angular para crear interfaces, como el backend, donde empleé Node.js y Express para construir una API.
Facilité la comunicación fluida entre el frontend y el backend... e Implementé la lógica de negocio del sistema de notas y activdades en el backend
